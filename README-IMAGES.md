# Image Setup Instructions

To complete the rewards page setup, please manually save the following images:

## Required Images:

1. **Appetizer Image**: Save the appetizer platter image as:
   - Path: `public/appetizer-platter.jpg`
   - This will be used for the "Free Appetizer" reward card

2. **Beverages Image**: Save the beverages image as:
   - Path: `public/beverages.jpg` 
   - This will be used for the "50% off Beverages" reward card

## Steps:
1. Right-click on each image from the conversation
2. Save them to the `public/` directory with the exact filenames above
3. The images will automatically appear in the rewards page

The code is already set up to display these images when they are properly saved to the public directory.
